# E-commerce TypeScript

Projeto desenvolvido para a atividade da **Aula 02 do TIC-HUB FullStack**.

## Tecnologias

* Node.js
* TypeScript

## Funcionalidades

* Cadastro de categorias
* Cadastro de produtos
* Sistema de usuários com roles
* Carrinho de compras
* Cálculo de total de itens
* Cálculo de valor final da compra

## Métodos utilizados

* `some()`
* `reduce()`

## Conceitos aplicados

* Programação Orientada a Objetos (POO)
* Interfaces
* Enum
* Tipagem forte
* High Order Functions

## Estrutura do projeto

```
src
 ├── index.ts
 └── model
      ├── category.model.ts
      ├── product.model.ts
      ├── user.model.ts
      └── cart.model.ts
```

## Executar o projeto

Instalar dependências:

```
npm install
```

Rodar o projeto em desenvolvimento:

```
npm run dev
```

ou

```
npx ts-node src/index.ts
```
